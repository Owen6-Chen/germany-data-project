Project 1: German Housing Market Analysis

# German Housing Market Analysis

## 1. Project Motivation
Finding affordable housing is a major challenge in many German cities.
This project analyzes rental housing data to understand price patterns
and build a simple price prediction model.

## 2. Main Questions
- How do rental prices differ across German cities?
- What factors influence rental prices?
- Can we build a simple model to predict rent?

## 3. Tools
- Python
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
